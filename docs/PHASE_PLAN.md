# Civitas-X 3-Phase Roadmap

## Phase 1 - Core (Current)

Goal: A convincing visual of a driverless-car city.

Deliverables:
- 2D city map with roads and route nodes
- autonomous car agents moving continuously
- simulation loop and rendering loop
- simple energy drain behavior

Definition of done:
- app launches and runs > 60 seconds without crash
- at least 20 cars can move independently
- visible route changes and stable frame updates

## Phase 2 - Expansion (NPC Agents)

Goal: Bring daily human life into the world.

Deliverables:
- NPC agents with identities and home/work anchors
- schedule system (time-of-day routines)
- trip requests to autonomous cars
- pickup/drop-off visualization

Definition of done:
- NPCs perform full daily cycle
- commute and work behavior is visible and repeatable

## Phase 3 - Simulation Layer (Economy + Life)

Goal: Make the city feel alive as a self-organized system.

Deliverables:
- points economy (salary, rent, charging costs)
- pricing model for rentals and energy
- aggregate metrics dashboard (mobility, utilization, earnings)
- balancing parameters for emergent behavior

Definition of done:
- complete points flow loop among NPCs and cars
- city behavior changes when economy variables are modified
